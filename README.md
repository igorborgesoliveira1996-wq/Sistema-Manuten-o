# Sistema de Manutenção

Sistema para gerenciamento de manutenção de equipamentos.

## Funcionalidades

- Cadastro de equipamentos
- Cadastro de técnicos
- Ordem de Serviço
- Histórico
- Dashboard

## Tecnologias

- HTML
- CSS
- JavaScript
- Python Flask
- SQLite

## Como executar

```bash
pip install -r requirements.txt
python app.py
```
